🧩 Where AI Should NOT Be Used (Critical for Interviews)

A senior engineer must show judgment.

AI should NOT:

• Make architectural decisions
• Replace domain experts
• Rewrite entire systems automatically
• Handle sensitive logic without human review
• Migrate code blindly without tests
